Laser Eye Surgery
=================

A game made with Unity in 48 hours for [Global Game Jam](https://globalgamejam.org/),
using the Tobii EyeX.

Theme: Rotation

[Entry Page](https://globalgamejam.org/2017/jam-sites/university-warwick).
